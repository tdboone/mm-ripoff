### L'il Vroom Vrooms

#### Q. What is this?

L'il Vroom Vrooms is a game inspired by Micro Machines for SNES.
It is a work in progress, but is reasonably fun even in its current state!

#### Q. How do I play it?

1. Be using Google Chrome
